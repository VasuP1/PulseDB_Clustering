
# PulseDB Clustering Project

This project involves clustering physiological time-series signals (ECG, PPG, ABP) using divide-and-conquer algorithms.

## Project Overview

- **Step 1**: Cluster Time-Series Segments
- **Step 2**: Closest Pair of Time Series
- **Step 3**: Apply Kadane’s Algorithm
- **Step 4**: Visualizations and Reports

## Installation

1. Install dependencies from `requirements.txt` using `pip install -r requirements.txt`.
2. Place your dataset in the `data/raw` folder.

## Usage

Run the `main.py` file to start the analysis.
